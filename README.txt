Завдання 1:
Напишіть програму, яка будує ряд кіл, центри яких лежать на 
колі. Число кіл задається при першому виклику рекурсивного методу.
Завдання 2:
Створіть додаток, що відображає рух окружності уздовж кордону вікна. 
Врахуйте можливість зміни розмірів вікна.
Завдання 3: 
Розробіть програму анімації руху на вибір (птаха, що летить, персонажа з 
мультфільму, чоловічка, тварини, ін. об'єкта.).
